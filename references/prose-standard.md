# Prose Standard

Use this standard for all working documents and final content. The target is clear strategy-consulting prose: direct, specific, economical, and useful to a decision-maker.

## Principles

### 1. Start With The Point

Lead with the answer, finding, or implication. Put setup after the point only when the reader needs it.

Weak:

```text
After reviewing several customer cohorts, we found differences in retention.
```

Stronger:

```text
Enterprise retention is the strongest reason to prioritize the segment.
```

### 2. Cut Filler

Remove phrases that add no meaning:

| Filler | Better |
|---|---|
| In order to | To |
| Due to the fact that | Because |
| It should be noted that | Cut |
| At this point in time | Now |
| A number of | Several |
| In terms of | Rewrite |

### 3. Prefer Active Voice

Use active voice unless the actor is unknown or irrelevant.

Weak:

```text
The decision was delayed by unclear ownership.
```

Stronger:

```text
Unclear ownership delayed the decision.
```

### 4. Be Specific

Replace vague claims with concrete nouns, metrics, and timeframes.

Weak:

```text
The initiative could create meaningful savings soon.
```

Stronger:

```text
The initiative could reduce support cost by 8 to 12 percent within two quarters.
```

### 5. Use Measured Confidence

State uncertainty plainly:

- High confidence: strong internal data and external validation
- Medium confidence: solid evidence with some assumptions
- Directional confidence: useful signal with limited proof

Avoid false precision.

### 6. Keep Sentences Lean

Use short sentences for key claims. Use longer sentences only when nuance matters.

### 7. Make Lists Parallel

If one bullet starts with a verb, the others should too.

```text
- Reduce approval thresholds for high-risk deals
- Equip account teams with value proof
- Track margin recovery by renewal cohort
```

### 8. Use Plain Business Language

Prefer:

| Avoid | Use |
|---|---|
| utilize | use |
| operationalize | implement |
| facilitate | help |
| robust | specific evidence |
| leverage as a verb | use |
| best-in-class | named benchmark |
| synergies | named benefit |

### 9. Make The Implication Visible

Every finding should answer one of these:

- What does this change?
- What decision does this support?
- What risk does this create?
- What action should happen next?

### 10. Edit Final Content Separately

For Stage 8, run a final pass:

1. Read only the headlines or section headings. Do they tell the story?
2. Cut setup language.
3. Replace vague adjectives with specifics.
4. Check that numbers have sources.
5. Make confidence levels explicit.
6. Read the executive summary aloud and tighten anything stiff.

## Words To Watch

Avoid or justify these:

- significant
- meaningful
- robust
- holistic
- strategic imperative
- key lever
- transformation journey
- unlock value
- game-changing
- seamless

Use the concrete claim instead.
